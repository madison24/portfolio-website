Meet

For this project I created an app called Meet, which users can view events in different cities.
Meet is a serverless, progressive web application (PWA) with React using a test-driven development (TDD) technique.
The application uses the Google Calendar API to fetch upcoming events.

PokeDex

PokeDex is an app that allows the user to select a pokemon name to pull up a modal with each pokemon height, weight, types and a picture.
The app also includes a contact form to send a message.
The app was built with Javascript, HTML, CSS and uses an external API.

myFlix API

myFlix API provides users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.
The api was built with Node.js and Express, rest architecture, body parsor, MongoDB, and more listed in package.json file.

myFlix App

myFlix is a movie app built using React based on existing server-side code I created previously - myFlix API.
The app allows users to view movies from an api, search movies by title, add or remove a movie from their favorites, select a movie to view more information, and update or delete their user profile.
myFlix app was built with MongoDB, Express, React, Node.js, Bootstrap, Parcel, and Proptypes.
